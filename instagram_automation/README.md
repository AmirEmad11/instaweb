# نظام الأتمتة العقارية على إنستجرام

## هيكل الملفات

```
instagram_automation/
├── gui.py               ◄ نقطة الدخول الرئيسية (الواجهة الرسومية)
├── bot_runner.py        ◄ منطق البوت (يعمل في Thread منفصل)
├── settings_manager.py  ◄ حفظ/تحميل الإعدادات من settings.json
├── session_manager.py   ◄ إدارة الجلسات وحفظ الكوكيز
├── lead_scraper.py      ◄ استخراج العملاء من التعليقات
├── automation_engine.py ◄ المتابعة، الرسائل، الرد على التعليقات
├── database.py          ◄ قاعدة البيانات SQLite
├── utils.py             ◄ أدوات مساعدة مشتركة
├── config.py            ◄ قيم افتراضية (لا تحتاج تعديلها يدوياً)
├── main.py              ◄ تشغيل بدون واجهة (CLI فقط)
├── requirements.txt     ◄ المكتبات المطلوبة
│
│   (تُنشأ تلقائياً عند التشغيل)
├── settings.json        ◄ إعدادات المستخدم (تُحفظ من الواجهة)
├── session_state.json   ◄ الجلسة المحفوظة (تسجيل دخول مرة واحدة)
├── leads.db             ◄ قاعدة بيانات العملاء
├── automation.log       ◄ سجل الأحداث
└── screenshots/         ◄ لقطات الأخطاء التلقائية
```

## الإعداد والتشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
playwright install chromium
```

### 2. تشغيل الواجهة الرسومية
```bash
python gui.py
```

### 3. من داخل الواجهة
1. أدخل **اسم المستخدم وكلمة المرور** ← اضغط "حفظ بيانات الدخول"
2. أضف **روابط المنشورات** في حقل الروابط ← اضغط "إضافة"
3. اضبط **الحدود اليومية والفواصل الزمنية** حسب الحاجة
4. اضغط **▶ بدء تشغيل البوت**

## الميزات

| الميزة | التفاصيل |
|--------|----------|
| واجهة رسومية | CustomTkinter - مظهر داكن احترافي |
| فصل Thread | البوت يعمل في خلفية منفصلة بدون تجميد |
| حفظ الجلسة | تسجيل دخول مرة واحدة فقط |
| استخراج العملاء | بحث بكلمات دلالية قابلة للتخصيص |
| رسائل ذكية | Spintax + قوالب متعددة |
| حماية من الحظر | تأخيرات عشوائية + كشف Action Block |
| قاعدة بيانات | SQLite لتتبع جميع العملاء |
| سجل مباشر | عرض الأحداث فور حدوثها مع ألوان |
| لقطات تلقائية | عند حدوث أي خطأ |
